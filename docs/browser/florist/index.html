Project overview

As the first project that was executed during Google UX Design course, I created Blooms - a florist app that help you buy flowers and customized them personally for your loved ones with just a few easy taps on the phone



TIMELINE

December 2021 - March 2022

WHAT I DID

Solo designer - user research, user flow, wireframe, hi-fi prototype, user testing, style guide

Competitor analysis: Opportunity and gaps

I analyzed 3 competitors that allow users to buy flowers online, while keeping previous user research in mind. The gap became great opportunity for our solution

The gap

The competitors solely focus on functional features and have little customization for users to feel more involved in the process of giving flowers

Project goal

Design a florist app that helps user easily order flowers and personally customize the gift for their beloved ones

1Personalized experience
2Florists credibility
2Quick and easy to use
